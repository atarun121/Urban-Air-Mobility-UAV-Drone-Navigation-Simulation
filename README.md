## 📌 Project Overview
[cite_start]As the landscape of **Urban Air Mobility (UAM)** evolves, the need for intelligent, collision-free navigation is paramount[cite: 71, 73]. [cite_start]This project features a robust simulation environment built in **MATLAB** designed to solve complex path-planning problems for UAV drones in high-density urban settings[cite: 73, 75].

[cite_start]By integrating **Genetic Algorithms (GA)** and **Particle Swarm Optimization (PSO)**, the system generates optimized trajectories that prioritize both safety and energy efficiency[cite: 74, 75].

---

## ✨ Key Features
* [cite_start]**3D Urban Modeling**: Designed high-fidelity environments to simulate realistic obstacles, ensuring the UAV can handle complex spatial constraints[cite: 73, 75].
* [cite_start]**Autonomous Path Planning**: Implemented a dual-optimization approach to calculate the shortest, most efficient route[cite: 74, 75].
* [cite_start]**Collision Avoidance**: Dynamic logic focused on maintaining safe flight corridors away from static and dynamic obstacles[cite: 74, 75].
* [cite_start]**Energy Optimization**: Integrated mission-specific constraints to minimize battery depletion during transit[cite: 75].

---

## ⚙️ Core Optimization Engine
[cite_start]The simulation utilizes two primary metaheuristic algorithms to ensure reliability and performance[cite: 74]:

1. [cite_start]**Genetic Algorithm (GA)**: Utilized for global exploration of the flight space to find the theoretical optimal path[cite: 74].
2. [cite_start]**Particle Swarm Optimization (PSO)**: Leveraged for real-time trajectory refinement and rapid responsiveness to environmental sensing[cite: 74, 76].
