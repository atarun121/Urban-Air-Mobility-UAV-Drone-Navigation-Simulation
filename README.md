Urban Air Mobility: UAV Drone Navigation Simulation
🚀 Project Overview
This project develops a high-fidelity simulation environment in MATLAB for autonomous drone navigation in complex urban settings. It addresses the challenges of Urban Air Mobility (UAM) by ensuring collision-free trajectories and optimizing for mission efficiency through advanced path-planning logic.

✨ Key Features
3D Environment Modeling: Custom-designed 3D maps to simulate realistic urban constraints including buildings and flight corridors.

Advanced Path-Planning: Implementation of metaheuristic algorithms to solve navigation challenges:

Genetic Algorithm (GA): Optimized for global path searching to identify energy-efficient flight corridors.

Particle Swarm Optimization (PSO): Utilized for real-time trajectory refinement and local obstacle avoidance.

Multi-Objective Optimization: Focuses on balancing the trade-offs between energy efficiency, shortest-path calculation, and safety against static and dynamic obstacles.

Sensing Integration: Navigation logic integrated with environmental sensing constraints to mimic real-world UAV responsiveness.

🛠️ Technical Stack
Language/Tool: MATLAB

Algorithms: Genetic Algorithm (GA), Particle Swarm Optimization (PSO)

Concepts: Path Planning, 3D Modeling, Autonomous Systems, Multi-objective Optimization
